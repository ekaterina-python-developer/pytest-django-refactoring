# YaNews - Новостной портал

## Описание
YaNews - это новостной сайт с системой комментирования. Пользователи могут читать новости, регистрироваться, оставлять комментарии и управлять ими.

## Основные возможности

### Для всех пользователей
- Просмотр 10 последних новостей на главной странице
- Чтение сокращенных версий новостей (первые 15 слов)
- Просмотр полного текста новости на отдельной странице
- Регистрация нового аккаунта
- Авторизация в системе

### Для авторизованных пользователей
- Добавление комментариев к новостям
- Редактирование своих комментариев
- Удаление своих комментариев
- Просмотр количества комментариев под новостью

### Особенности системы
- Фильтрация запрещенных слов в комментариях
- Защита от редактирования чужих комментариев
- Новости отсортированы от новых к старым
- Комментарии отсортированы от старых к новым

## Технологии
- Python 3.x
- Django
- pytest
- SQLite3
- unittest

## Тесты

Проект включает три группы тестов:

### test_routes.py
- Доступность главной страницы
- Доступность страниц новостей
- Контроль доступа к редактированию комментариев
- Редиректы для анонимных пользователей
- Доступность страниц аутентификации

### test_content.py
- Не более 10 новостей на главной
- Правильная сортировка новостей
- Правильная сортировка комментариев
- Доступность форм комментариев

### test_logic.py
- Отправка комментариев
- Фильтрация запрещенных слов
- Управление своими комментариями
- Защита от изменения чужих комментариев

## Установка и запуск

# Клонирование проекта
git clone git@github.com:dariazueva/ya_news.git
cd ya_news

# Создание виртуального окружения
python -m venv env
source env/bin/activate  # Linux/macOS
# или
env\Scripts\activate     # Windows

# Установка зависимостей
pip install -r requirements.txt

# Настройка базы данных
python manage.py migrate
python manage.py loaddata news.json

# Запуск тестов
pytest
